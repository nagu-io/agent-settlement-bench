# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch and current release tag.

## Reporting a Vulnerability

Please do not open public issues for security reports.

Use a private GitHub vulnerability report:
- https://github.com/nagu-io/agent-settlement-bench/security/advisories/new

Include:
- affected file or script
- reproduction steps
- expected vs actual behavior
- impact assessment (data loss, incorrect settlement decision, remote code risk, etc.)

## Response Targets

- Initial triage: within 3 business days
- Status update: within 7 business days
- Patch timeline: depends on severity and reproducibility

## Scope Notes

This repository is a benchmark framework. It does not manage private keys, signing, or production custody infrastructure.
